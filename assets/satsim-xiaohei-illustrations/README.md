# SatelliteSimJulia Xiaohei Illustration Set

This directory stores Ian-style Xiaohei illustrations for SatelliteSimJulia.

## Visual Contract

- 16:9 horizontal Chinese article illustration.
- Pure white background.
- Minimal black hand-drawn line art.
- Sparse red/orange/blue handwritten notes.
- Xiaohei must perform the core conceptual action.
- One image explains one idea only.
- Avoid formal architecture diagrams, PPT infographics, dense charts, cute mascots, or UI screenshots.

## Naming

Use stable numeric prefixes:

```text
001-overview-time-slicer.png
002-foundation-constants.png
...
```

The original generated images remain under Codex's generated image directory. Project copies live here.
